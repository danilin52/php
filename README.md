# Лабораторная работа: Классы и Объекты в PHP

## Описание классов
- **User**: Базовый класс для хранения данных пользователя (имя, логин, пароль). Содержит конструктор с продвижением свойств и деструктор.
- **SuperUser**: Наследник класса User, добавляющий информацию о роли пользователя в системе.
